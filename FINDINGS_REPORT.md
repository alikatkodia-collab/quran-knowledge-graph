## Methodology

### Data Source
Rashad Khalifa's English translation of the Quran (*The Final Testament*), 6,234 verses across 114 surahs.

### Knowledge Graph
- TF-IDF keyword extraction with lemmatization (2,674 keywords)
- Verse-to-verse edges via shared rare keywords (106,393 edges)
- Parameters optimized via Karpathy-style AutoResearch loop

### Deduction Engine
- **Proposition extraction**: spaCy NLP extracts subject-verb-object triples
- **Syllogistic rules**: Transitive chains (A→B→C), shared-subject synthesis, 3-hop thematic bridges
- **Novelty scoring**: Graph distance × cross-surah diversity × bridge keyword uniqueness

### Analysis Loops (9 concurrent)
1. Deduction generation (syllogistic axiom combination)
2. Categorization + meta-knowledge graph building
3. Meta-graph parameter optimization
4. Contradiction/tension detection
5. Deep chain extension (4-5 hops)
6. Surah narrative arc analysis
7. Rhetorical structure detection (ring composition, repetition)
8. Intertextuality analysis (parallel passages, cross-references)
9. Statistical linguistics (vocabulary, symmetries, co-occurrence)
